# platform-public-docs

letsur 플랫폼의 **공개 문서** 저장소입니다. 플랫폼 UI에서 `react-markdown` 으로 직접 렌더링하여 사용자에게 제공합니다.

## 구조

```
guide/        API 사용 가이드 등 개발자 대상 문서
legal/        이용약관·개인정보 처리방침 등 법적 문서
```

## 문서 목록

### Guide
- [`guide/api-guide.md`](./guide/api-guide.md) — API 사용 가이드 (Overview)

### Legal
- [`legal/terms-of-service.md`](./legal/terms-of-service.md) — 서비스 이용약관
- [`legal/privacy-policy.md`](./legal/privacy-policy.md) — 개인정보 처리방침
- [`legal/marketing-consent.md`](./legal/marketing-consent.md) — 마케팅 수신동의

## 기여

각 문서는 별도 PR로 관리합니다. 문서 변경이 필요하면 브랜치를 파고 PR을 올려주세요.
